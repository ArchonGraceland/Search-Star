# Search Star

**Sovereign Personal Data Standard — Own your data, set your price.**

Search Star is an open-source standard for self-sovereign personal profiles. You host your own data, platforms query your API, and you get paid per query.

## Site Structure

| Page | Description |
|------|-------------|
| `index.html` | Home page — profile directory, stats, create-profile flow, spec links |
| `profile.html` | Sample profile (SS-000001: Steve Smith) — all data sections, schema, monetization |
| `spec.html` | Full specification document — schema, Presence Composite, access control, roadmap |

## What It Is

A structured, machine-readable personal data profile covering:

- **Financial Standing** — Net worth, income, credit, savings as age-cohort percentiles. Verified via zero-knowledge proofs.
- **Presence Composite** — Rizz, Vibe, and Drip. Deliberately brand-blind.
- **Skills & Credentials** — W3C Verifiable Credentials.
- **Interests** — Athletic, Social, and Intellectual domains.

## Three Access Tiers

| Tier | Access | Pricing |
|------|--------|---------|
| Public | Open to all | Per query (owner-set) |
| Private | Owner-approved only | Per query (owner-set) |
| Marketing | Open to all | Per message (owner-set, no refunds) |

## Profile Numbering

Profiles are assigned permanent sequential IDs: SS-000001, SS-000002, etc.
